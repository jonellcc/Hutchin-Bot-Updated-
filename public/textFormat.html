Hutchin Bot 2.0 - textStyle Feature

Overview

The textStyle feature in Hutchin Bot 2.0 is designed to enhance the formatting of messages by automatically applying bold text styling to the Title and Footer. Additionally, any text wrapped with **bold text** within the Description field will be automatically rendered as bold.

How It Works

  • Title: Always rendered in bold text.
  • Footer: Always rendered in bold text.
  • Description: If any part of the Description is wrapped in **bold text**, that section will be rendered in bold.

Example Input

{
    Title: "**AI Response**",
    Description: "Hi again! **How's your day** going so far?",
    Footer: "Powered by Zetsu | 🕰️ 4/27/2025, 8:14:28 AM"
}

Implementation Steps

To integrate the textStyle functionality in your bot:

  1. Define a message object with Title, Description, and Footer fields.
  2. Use the textFormat.js function to format the message text based on the input.
  3. Ensure that the Title and Footer are always rendered in bold, and any bolded sections in the Description are automatically formatted.
  4. Integrate the formatted message into your bot's response system.

Conclusion

The textStyle feature in Hutchin Bot 2.0 is designed to give your messages a polished look by automatically applying bold formatting to specific sections like the Title and Footer, while allowing flexibility in the Description. It's easy to implement and enhances the clarity and aesthetics of bot responses.
